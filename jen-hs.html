Hardware vs. Software
is a type of computer memory that can be accessed randomly; that is, any byte of memory can be accessed without touching the preceding bytes.

The difference



One key difference is that hardware is physical while software is not. Hardware is any device that is used externally/physically with one's device. Software, on the other hand is dependent on hardware in order to operate. It always relies on one or more hardware device. A game, for example, uses CPU (processor), RAM (memory), hard drive, and a video card, all externally installed on the computer as hardware.
